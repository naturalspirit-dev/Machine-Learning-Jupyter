## Web-app generating text based on a trained Recurrent Neural Net (RNN) model

### Requirements
* keras
* TensorFlow
* Numpy
* wtforms

---

### How to run
Clone/copy all the files in this folder into a single folder on your computer and run

**`$ python keras_server.py`**
